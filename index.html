WHITEPAPER


The Blockchain of Tezos is currently in the search for development projects that help to form its ecosystem and, as expected, as has always been the case with other blockchain, the needs within the ecosystem are very fundamental for the Blockchain de Tezos enjoy a standard usability. On the part of enthusiastic communities of blockchain technology, one of the necessary implementations is the development of web applications that help the blockchain developer of Tezos to create, debug and run their smart contracts in a practical, reliable and secure manner.

This is where the Skyrius proposal was born, “a web application that allows the creation, editing, debugging and deployment of smart contracts in MICHELSON”.

Skyrius... why Skyrius? The name of Tezos comes from the Lithuanian, that in Spanish would be Thesis or group of Theses, taking into account that, “each valid contract would be then, chapters of this thesis, from there Skyrius is born”. The project to be open source and web has two high-level language proposals that will be linked to the language of intelligent contracts in MICHELSON.

1.- C# (Core.net, to run on Linux) and Javascript as a high level language.
2.- JAVA and Javascript as high level languages.
